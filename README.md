# Hello World App

A simple Hello World application built with Expo and React Native.

## Features

- Clean and modern UI
- Centered "Hello, World!" display
- Built with Expo and React Native

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npx expo start
   ```

## Technology Stack

- Expo
- React Native
